# NSGABLACK框架中文教程

本教程系列提供了NSGABLACK优化框架的全面学习资源，从基础概念到高级应用。

## 教程结构

### 基础篇（0-6）
- **[00_项目概述与架构介绍](00_项目概述与架构介绍.ipynb)**
  - 框架简介和设计理念
  - 核心模块概览
  - 安装和快速开始

- **[01_偏置系统核心概念详解](01_偏置系统核心概念详解.ipynb)**
  - 偏置系统基本原理
  - 偏置类型和作用机制
  - 基础偏置实现

- **[02_创建自定义偏置](02_创建自定义偏置.ipynb)**
  - 自定义偏置开发
  - 偏置组合策略
  - 实际案例分析

- **[03_蒙特卡洛优化](03_蒙特卡洛优化.ipynb)**
  - 随机搜索原理
  - 蒙特卡洛方法实现
  - 性能优化技巧

- **[04_NSGAII多目标优化](04_NSGAII多目标优化.ipynb)**
  - 多目标优化基础
  - NSGA-II算法详解
  - 帕累托前沿分析

- **[05_贝叶斯优化](05_贝叶斯优化.ipynb)**
  - 贝叶斯优化理论
  - 高斯过程模型
  - 采集函数设计

- **[06_变邻域搜索VNS](06_变邻域搜索VNS.ipynb)**
  - VNS算法原理
  - 邻域结构设计
  - 局部搜索策略

### 进阶篇（7-13）
- **[07_算法嵌套与混合策略](07_算法嵌套与混合策略.ipynb)**
  - 多算法协同框架
  - 嵌套优化策略
  - 自适应切换机制

- **[08_代理模型详解与应用](08_代理模型详解与应用.ipynb)**
  - 代理模型类型
  - 模型训练和更新
  - 计算资源优化

- **[09_NSGABLACK工具集详解（一）：核心基础工具](09_NSGABLACK工具集详解（一）：核心基础工具.ipynb)**
  - 精英管理系统
  - 问题定义框架
  - 核心数据结构

- **[10_NSGABLACK工具集详解（二）：实用工具与并行计算](10_NSGABLACK工具集详解（二）：实用工具与并行计算.ipynb)**
  - 并行评估器
  - 内存管理器
  - 实验管理系统

- **[11_NSGABLACK框架功能总览与实战应用](11_NSGABLACK框架功能总览与实战应用.ipynb)**
  - 框架功能整合
  - 实战项目演示
  - 最佳实践指南

- **[12_偏置-代理-算法三重协同优化](12_偏置-代理-算法三重协同优化_最终版.ipynb)**
  - 三重协同机制
  - 动态调整策略
  - 高级优化技术

- **[13_并行计算与性能优化](13_并行计算与性能优化.ipynb)**
  - 并行化策略
  - 性能分析工具
  - 优化技巧总结

### 高级篇（14-15）
- **[14_NSGABLACK框架架构总览](14_NSGABLACK框架架构总览.ipynb)**
  - 框架设计哲学
  - 模块间交互机制
  - 扩展性和可维护性

- **[15_偏置驱动的约束优化与高级应用](15_偏置驱动的约束优化与高级应用.ipynb)**

  - 贝叶斯优化偏置系统

  - 偏置-代理-算法三重协同机制

## 学习路径建议

### 初学者路径
1. 从 **00_项目概述与架构介绍** 开始，了解框架整体
2. 学习 **01-03** 掌握基础概念
3. 通过 **04-06** 了解核心算法

### 进阶用户路径
1. 复习基础篇内容
2. 深入学习 **07-10** 的高级特性
3. 实践 **11-13** 的综合应用

### 研究者路径
1. 快速浏览 **00-06**
2. 重点研究 **07-13** 的高级技术
3. 深入学习 **15** 的偏置驱动创新机制
4. 从 **14** 获取架构设计灵感
