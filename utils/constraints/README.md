# utils/constraints

## Purpose
Constraint helper and repair utilities.

## Trigger Timing
Called by pipeline/problem constraint stages.

## Input
- Candidate solutions, bounds, constraint parameters.

## Output
- Repaired solutions and/or violation values.

## Side Effects
May mutate arrays in place.

## Failure Policy
Fail with clear diagnostics if repair contract is broken.
