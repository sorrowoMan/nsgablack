# ExampleSurrogateBias

Template for a surrogate control bias.

## Purpose
- Show how to implement `SurrogateControlBias`.
- Demonstrates a simple phase switch via `context.progress`.

## File
- `bias/surrogate/template_surrogate_bias.py`

## Usage
Pass the instance into `SurrogateUnifiedNSGAII(..., surrogate_biases=[...])`.
